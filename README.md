Opak Sans
============

Opak Sans is a tweaked version of Intel's 'Clear Sans'. Opak Sans has been paired down and adjusted on the Gnu+Linux desktop to work well with the Freetype font render engine. The Opak fonts are released under the Open Font License with full and free access to the project's fonts and source files via Github.
